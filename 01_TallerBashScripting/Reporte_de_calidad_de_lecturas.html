Tablas de identifiacin de casos

Algo que es muy til en estos reportes es la capacidad de mostrar u ocultar cdigo. A continuacin se mostrarn tablas realizadas con la siguiente estructura de cdigo.

# dataframe filtrado para pruebas aprobadas
df <- qc %>% 
  select(Muestra, Mdulo, Estatus) %>%    
  filter(Estatus %in% c("APROBADO")) %>%
  arrange(Muestra)

# parametros graficos para impresin de tabla
kable(df, "html") %>%
  kable_styling(
    font_size = 15,
    full_width = F,
    position = "left",
    bootstrap_options = c("striped", "hover")
  ) %>%
  row_spec(0, bold = T, color = "white", background = "darkgreen")
